1. Web Aplikasi Manajemen Tugas – TaskFlow.

Aplikasi berbasis web untuk mengelola tugas harian dan kerja tim secara kolaboratif. Fitur: drag & drop, komentar, dan pengingat otomatis.

2. Aplikasi Mobile Kesehatan – FitCheck.

Aplikasi Android/iOS untuk mencatat kalori, memonitor aktivitas fisik, dan memberikan rekomendasi olahraga ringan harian.

3. Sistem Informasi Sekolah – EduTrack.

Sistem informasi untuk mengelola data siswa, jadwal pelajaran, dan laporan nilai dengan dashboard admin dan akses guru/siswa.

4. Dashboard Analitik Penjualan – SalesViz

Dashboard visualisasi data untuk toko online berbasis e-commerce. Menampilkan grafik penjualan, tren produk, dan analisis pelanggan.

5. Chatbot Customer Service – HelpBot AI

Chatbot cerdas yang dapat menjawab pertanyaan pelanggan secara otomatis, terintegrasi dengan WhatsApp dan website.

6. Game Edukasi Anak – MathHero.

Game interaktif untuk anak-anak usia 6–10 tahun yang mengajarkan matematika dasar lewat misi dan level.
